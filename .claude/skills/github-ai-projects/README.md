# GitHub AI热门项目监控 Skill

这是一个专门用于监控GitHub上AI领域热门开源项目的技能，能够提供项目的简述信息和不同时间范围的更新状态分析。

## 功能特点

- 🔍 **智能搜索**：基于GitHub API搜索AI相关项目
- 📊 **多维度分析**：星标、活跃度、社区参与度等综合评估
- ⏰ **时间范围**：支持最近1天、本周、最近一个月的更新状态
- 📝 **项目简述**：为每个项目生成简洁明了的描述
- 🏆 **热度排行**：基于多个指标的综合热度评分

## 使用方法

在对话中输入以下任一指令：

- "获取GitHub热门AI项目"
- "查看最近AI项目更新状态"
- "分析本周AI项目活跃度"
- "GitHub上有什么新的AI项目"

## 输出格式

技能会按照以下格式提供信息：

1. **热度排行榜**：Top 10热门项目
2. **时间维度分析**：不同时间范围的活跃项目
3. **重点推荐**：综合评估的优秀项目
4. **数据洞察**：整体趋势分析

## 技术实现

- 基于GitHub搜索API
- 多维度项目评估算法
- 智能项目简述生成
- 时间序列活跃度分析

## 注意事项

- 遵循GitHub API调用限制
- 确保数据时效性和准确性
- 保护用户隐私信息
- 提供客观的项目评估
